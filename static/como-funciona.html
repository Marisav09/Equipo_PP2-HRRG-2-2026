¿Cómo funciona?

Guía rápida de uso

Dos perfiles, una misma idea: ingresar con el perfil correspondiente y hacer consultas claras sobre equipos médicos.

Si la información no existe

El sistema indicará que no dispone de información suficiente para responder la consulta.

Perfil

Operador

1

Ingresar

Seleccione "Ingresar como Operador". Si escaneó un QR, luego del ingreso se abrirá directamente ese equipo.

2

Consultar

Escriba o dicte una pregunta relacionada con el funcionamiento del equipo.

3

Leer

El sistema mostrará una respuesta clara y sencilla basada en documentación disponible.

Ejemplos

¿Cómo silenciar una alarma? ¿Cómo iniciar una prueba? ¿Qué significa este código?

Perfil

Técnico

1

Ingresar

Seleccione "Ingresar como Técnico" y acceda con credenciales autorizadas. Si escaneó un QR, se abrirá directamente ese equipo.

2

Consultar

Realice preguntas sobre procedimientos, configuraciones, alarmas o mantenimiento.

3

Validar

Revise las referencias a manuales y páginas utilizadas para generar la respuesta.

4

Usar herramientas

Acceda a consulta avanzada, historial, equipos, códigos QR y documentación asociada.
